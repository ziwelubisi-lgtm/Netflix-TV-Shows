
# 📺 Netflix TV Shows Dataset & Insights

This repository contains a curated dataset of popular TV shows, including their release year, age rating, IMDb rating, and availability on Netflix.  
It also provides analytical insights you can explore to understand trends in television content.

---

## 📂 Dataset Overview

The dataset (`TV Shows - Netflix.csv`) includes the following columns:

- **Titles** → Name of the TV show  
- **Year** → Release year  
- **Rating** → Age rating (e.g., 7+, 16+, 18+)  
- **IMDB_Rating** → IMDb rating (scale 1–10)  
- **Netflix** → Availability flag (1 = available on Netflix, 0 = not available)

---

## 🔎 Insights You Can Explore

- **Netflix Strengths** → Netflix hosts several top-rated shows (*Breaking Bad*, *Sherlock*, *Avatar: The Last Airbender*).  
- **Catalog Gaps** → Some classics (*Game of Thrones*, *The Wire*) are missing from Netflix.  
- **Audience Focus** → Majority of Netflix shows are rated **16+ or 18+**, targeting mature viewers.  
- **Quality Comparison** → Compare average IMDb ratings of Netflix vs non-Netflix shows.  
- **Time Trends** → The 2010s dominate the dataset, reflecting the “golden age of TV.”  
- **Hidden Gems** → Discover underrated but high-rated shows (*Mindhunter*, *Narcos*, *Peaky Blinders*).
